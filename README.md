# Office Image Extractor

A simple Python GUI application to extract original images from Microsoft Office documents (.pptx and .docx).

## Features

- **Original Quality**: Extracts images directly from the document structure without re-encoding.
- **Drag & Drop**: Simply drag your files onto the application.
- **Batch Processing**: Automatically creates folders for extracted images.
- **Format Support**: Supports standard image formats and vector graphics (EMF, WMF) stored within Office files.
